###  Stack Tecnológico
* **Lenguaje:** Java 11
* **Herramienta de Automatización:** Selenium WebDriver
* **Framework de Pruebas:** TestNG
* **Gestor de Dependencias:** Maven
* **Reportes:** Allure Framework
* **Integración Continua (CI):** GitHub Actions

###  Características del Proyecto
* **Patrón de Diseño:** Implementación básica de Page Object Model (POM).
* **Ejecución en la Nube:** Configurado para correr en contenedores de Ubuntu mediante GitHub Actions.
* **Modo Headless:** Optimizado para ejecuciones en entornos de Integración Continua sin interfaz gráfica.
* **Manejo de Reportes:** Generación automática de historial de pruebas y capturas de pantalla en caso de error.
